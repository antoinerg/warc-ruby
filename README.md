# Web ARChive for Ruby

Please see this post for announcement: https://antoineroygobeil.com/blog/2013/10/25/warc-ruby/

## Installation

Add this line to your application's Gemfile:

    gem 'warc'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install warc
